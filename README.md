mpeg2ts
=======

This tool analyses HLS *.ts files to get containing streams' info.

Run it as:
> ruby mpeg2ts.rb [ts_name] [-v]

ts_name - name of HLS fragment you need to analyze 

-v - verbose output

See output examples in Wiki.
